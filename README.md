# xyw 在线工具

这个仓库放置了多个可以直接访问的网页工具，都是纯前端页面，不需要安装软件。

## 编程语言关键词训练场

在线地址：

[https://nuyoahxyw.github.io/xyw/](https://nuyoahxyw.github.io/xyw/)

用于学习和复习 C#、C、C++ 与 Python 的语言关键词。每个词条包含读音、中文解释、代码示例和容易混淆的知识点，并提供随机刷题、分类筛选、关键词典与错题复习。

题库范围：

- C#：122 个特殊词汇
- C17：44 个标准关键字
- C++20：92 个保留关键字与 4 个上下文标识符
- Python 3.13：35 个保留关键字与 4 个软关键字

特点：

- 支持四种编程语言快速切换
- 浏览器原生英文朗读与音标提示
- 学习进度和错题按语言独立保存在本机
- 纯前端离线运行，手机与电脑均可使用

## 音视频格式转换器

在线地址：

[https://nuyoahxyw.github.io/xyw/audio-converter.html](https://nuyoahxyw.github.io/xyw/audio-converter.html)

用于在浏览器里进行音频、视频格式转换。页面支持拖入文件，本地处理后下载转换结果，适合临时把常见音视频文件转换成更方便使用的格式。

特点：

- 浏览器本地处理，不需要上传服务器
- 支持音频转 MP3
- 支持常见视频文件转音频或转换处理
- 适合日常学习、剪辑、资料整理时快速使用

## 思政课程刷题库

在线地址：

[https://nuyoahxyw.github.io/xyw/sizheng-quiz.html](https://nuyoahxyw.github.io/xyw/sizheng-quiz.html)

用于大学思政课程期末复习刷题。页面内置多门课程题库，每轮从当前课程完整题库中随机抽取 60 道题，答完后自动按 100 分制出分，并显示本轮错题解析。

已包含课程：

- 习近平新时代中国特色社会主义思想概论
- 毛泽东思想和中国特色社会主义理论体系概论
- 中国近现代史纲要
- 马克思主义基本原理
- 思想道德与法治

特点：

- 每门课程都有 300+ 高频考点题库
- 每轮强制完成 60 道题
- 点选后立即判断正误
- 做完后显示分数和本轮错题解析
- 手机浏览器也可以直接使用

> 题目为学习练习用途，依据公开教材知识点和课堂常见考点整理。考试表述请以任课教师、教材和学校题库为准。
